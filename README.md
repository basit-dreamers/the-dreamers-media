# The Dreamers Media

An immersive 3D landing experience for a modern creative marketing agency.

## Files

- **`index.html`** — the main 3D immersive experience (WebGL + Three.js)
- **`styles.css`** — styles for the site
- **`script.js`** — WebGL scene logic and interactions

## Features

- WebGL scene with animated icosahedron core, orbital rings, floating satellites, and a 2,500-star field
- Scroll-driven camera choreography across 6 keyframed scenes
- Mouse parallax, custom cursor, scroll progress bar
- Fully responsive, no build step required — just open the HTML file

## Stack

Vanilla HTML / CSS / JS · [Three.js](https://threejs.org) via CDN import map.

## Local preview

Just open `index.html` in a modern browser.
